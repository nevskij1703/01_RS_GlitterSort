# GlitterSort — Колбочки

Single-file HTML-игра «Колбочки» (Water Sort Puzzle). Цель — рассортировать цветные слои так, чтобы каждая колба содержала только один цвет.

## Особенности

- **10 ручных уровней** с возрастающей сложностью + бесконечный режим начиная с 11-го (процедурная генерация по тем же алгоритмам).
- **3–12 колбочек** на уровне, **2–4 цвета** максимум.
- **Ход назад** — откатывает последний переливание.
- **+ Бонусная колбочка** — за просмотр рекламы добавляется ещё одна пустая колба (один раз на уровень).
- **Локальное сохранение** прогресса (`localStorage`), уровень детерминированно восстанавливается по seed.
- **Дев-панель** (`Ctrl+Shift+D`): живая настройка параметров уровня — количество колб, ёмкость, цвета, целевое число ходов, режим без тупиков.
- **Реклама** через Yandex Mobile Ads SDK (interstitial между уровнями + rewarded за бонусную колбу). Подробности интеграции в [`docs/ADS.md`](docs/ADS.md).
- **Один файл** `index.html` без зависимостей — открывается двойным кликом, легко упаковывается в APK.

## Быстрый старт

1. Открой `index.html` в браузере — игра начинается сразу с первого уровня.
2. Клик по колбочке → выделение → клик по другой → переливание (если цвет верхнего слоя совпадает или приёмник пуст).
3. Когда все колбочки моноцветные или пустые — уровень пройден.

## Сборка APK

1. Открой Claude Code в этой папке.
2. Запусти скилл `build-apk-from-html` (или попроси: «собери apk»).
3. Подключи Yandex Mobile Ads SDK в Android-обёртке по инструкции из [`docs/ADS.md`](docs/ADS.md).
